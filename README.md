#丝绸之路招募书

###副标题：一群流民还是流民，只有共同的目标，他们才可能形成组织，更进一步才是文明

##我有一个梦想

和一群可爱的人做一件大家喜欢的事情，这群可爱的人有几个基本属性，1.是有爱的 2.是有社会责任感的 3.是有理想的 4.是敢于实践的。我找到了一群天天和生命打交道的人，一群种地和养鸡的人，我们来自五湖四海，为了一个共同的目标走到一起，一起享受自己亲手制作的食物，在六环边上挥汗如雨的干活。美好生活的背后我发现大家的收入很低，工作时间比较长，我们想做一份可持续的事业，但是产出和销售总是脱节，经济上的不可持续导致人才的流动性很大。

##建立属于我们自己的物流体系

目前大部分农场现有的物流方式就是普通快递和专线配送，这两种方式部分解决了会员制业务和零售业务的配送需求。但同时也极大程度的限制业务模式的创新，由于生鲜食品普遍存在体积大，对温度敏感，特别是对温度变化的敏感，导致没有经过专业培训的常规派送业务员很难胜任生鲜食品的派送服务。

组建一个属于社员共有的物流合作社是行业良性发展的基础保障，同时也是面对强大资本力量之下增强小农场的生存能力和竞争力的后盾。农场之间打通物流通道将给行业带来巨大发展空间，是增强农场直销的有力方法。

###合作社设计的几个基本原则：

    自由加入，遵守社章；

    民主管理；

    坚持劳动者主权；

    社员参与企业管理；

    合作社之间的合作；

    报酬的一致性；

    发展教育；

##调整生产结构

农场是这个行业普遍存在的经营模式，它的主要功能是种植和养殖，大部分农场都是通过租地雇工，自产自销的方式，为了满足消费者的长期需求和维持农场周年运转，生产者往往需要计划多个品种的产品。因为发展速度和经营规模的限制，同时生鲜产品本身固有的时效性，导致农场的产品种类多，单产规模小，单位产出投入高，集约化程度低，专业化经验积累比较困难，产品的商品率普遍偏低。这些结构上的不合理并不是设计者当初期望的结果，主要是从业者对农业生产的复杂性和风险性判断不足。但是这并不能否认这种生产结构对行业起步阶段所起到的建设性作用，它的出现是建立在消费者对常规农产品出现信任危机的前提下诞生的。当生产者在没能找到同类生产标准的产品情况下，他们只能选择更多的自产品。大家找到更多的同类生产者是自身生产结构升级的基础保障。

我们生活中的农产品大部分有明显的地域特色，北京周边的生产是不能完全满足消费者的需求，同时什么都生产也不是特别经济的方式。发展阜外同类生产者是消费者需求导向的理性方案，阜外产品的入京销售能有力提升销售产品的丰富度和客户的满意度，同时降低生产成本和农场的管理复杂度。

##技术进步

农业生产的一个趋势性变化是从原来的劳动密集向技术密集转变，作为生态友好型农产品行业也逃不脱这个规律的制约。农业技术的进步不但需要发明者，同时也需要传播者和使用者。如果行业不能触发新的技术变革，新技术不能找到生存空间，那么整个行业的良性发展会最终受到限制。像良种化这种能都提升整个行业竞争能力的关键性技术，需要长期投入并不断累积，它不是某个农场通过多元化生产能够达到的。一些专业化生产的合作社和家庭农场能有更好的经验积累，我们需要和他们建立广泛的合作关系。

##把节约提升到和生产并重的地位

一个新入行不久的新人首先会感觉我们的事业真美好，等过了实习期后他发现我们干的事情还不够专业，或者离他当初的美好有差距。当看到自己亲手劳动的收获被各种损耗浪费，出现情绪是可以理解的。管理者面对各种抱怨和意见需要认真分析实际情况，部分问题是现在的业务模式和经营结构所固有的。比如说每周五天的配送日，同样是准备60箱蔬菜在一个工作日内完成，一件事情只需要做一次，分到五个工作日内完成，一件事情要重复做5次，农民花在路上的时间就多出4倍来，生产中的磨洋工现象和业务模式和管理方式紧密相关。但是你让大家都按照生产的节奏来开展业务，明显也是不合理的，产品最后需要出售才能获得盈利，诺大一个北京城，消费者遍布全北京市，怎么能及时配送到消费者手中是良好销售的关键环节，在物流保质保量的前提下才有可能灵活的调整业务模式。通过集中采摘收获和集中配送能够节约更多劳动力，这就是村里人常说的“出活”。

##改善包装和储运

环境保护的理念深入大部分从业人员，这样的理念是非常正确的，值得大家的推广和坚守。但是目前各家的包装在设计上还不便于物料回收，很多农场甚至采用大量的泡沫保温箱，最终导致一件充满社会价值的事情却需要较高的成本。这个方案最终没有带来理想的经济效益，另一个主要原因是周转箱的丢失情况异常严重，一个远没有达到报废时限的周转箱早已在某次配送过程中人为的被消失。其实最好的方式是农场统一租用物流公司的包装箱，各自发挥各自的特长，专业互补，每一各环节保证有利益方持续的投入和改进。

同样的问题出现在农产品的保存上，因为大部分农场都从小规模开始，大家都没有足够的资金投入仓储建设中，同时食品保存的特殊性对仓储建设的专业性提出较高要求，防鼠防虫，换气调温。同时来自不同地区的农产品运到北京后，很多情况会出现水土不服现象，比如来自北方的土豆入京后保存条件不够极易导致发青发芽，东北的大米普遍含水量偏高，容易发生霉变。生态友好型产品需要执行严格生产标准，同时也需要保证生态友好的储运。

##促进行业标准

一个不能总结自身标准的行业，业内之间的竞争将会在很长时间内继续比较低级的嘴仗，谁也不服谁，或者大家相互之间讳莫如深，要不大家在网上骂骂咧咧一通，无助与行业的进步。我觉得行业标准的起步建立在各生产者之间的互通有无，我认为这里分几个层次，首先物流通路能促进行业升级，人才流入促进行业稳定，其次信息流动促进生成行业标准，最后的群策群力形成产业竞争。
